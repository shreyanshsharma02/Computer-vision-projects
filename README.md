# Computer Vision Projects

A collection of beginner to intermediate Computer Vision projects built using Python and OpenCV.

This repository documents my journey of learning Computer Vision, starting from webcam basics and progressing towards real-time AI applications.

---

## 📂 Projects

### 1. Webcam Basics
- Capture webcam feed
- Display live video
- Learn how OpenCV interacts with cameras

**Skills**
- cv2.VideoCapture()
- cv2.imshow()
- cv2.waitKey()

---

### 2. Face Detection

Detect human faces in real time using Haar Cascade classifiers.

**Skills**
- Haar Cascades
- detectMultiScale()
- Image preprocessing

---

### 3. Finger Counter

Count fingers in real time using hand landmark detection.

**Skills**
- MediaPipe
- Hand Tracking
- Landmark Detection
- Gesture Recognition

---

## 🛠️ Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy

## Installation

Clone the repository

```bash
git clone https://github.com/shreyanshsharma02/Computer-vision-projects.git
cd Computer-vision-projects
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run any project

```bash
cd 01-webcam-basics
python webcam.py
```
